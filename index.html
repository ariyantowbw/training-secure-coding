Lab Secure Coding - CyberAcademyID

Lab Secure Coding dirancang untuk mengajarkan pengembang dan profesional keamanan cara mengenali, menganalisis, dan mengatasi kerentanan pada aplikasi perangkat lunak.

SQL Injection

jenis kerentanan di mana penyerang dapat menyisipkan perintah SQL berbahaya melalui input pengguna yang tidak tervalidasi, untuk mengeksploitasi basis data aplikasi.

Total: 2 Lab
NoSQL Injection

jenis kerentanan di mana penyerang dapat menyisipkan perintah No SQL berbahaya melalui input pengguna yang tidak tervalidasi, untuk mengeksploitasi basis data aplikasi.

Total: 1 Lab
Authentication

Jenis kerentanan dimana penyerang dapat memanipulasi proses otentikasi.

Total: 3 Lab
LFI (Local File Inclusion)

Jenis serangan di mana penyerang dapat memanfaatkan kerentanannya untuk menyertakan file lokal yang ada di server web

Total: 2 Lab
File Upload

jenis kerentanan di mana penyerang dapat mengupload file yang tipe ekstension apapun.

Total: 2 Lab
Cross Site Scripting (XSS)

jenis kerentanan di mana penyerang dapat menyisipkan perintah berbahaya yang dapat mencuri cookie korban.

Total: 1 Lab
Insecure Direct Object Reference (IDOR)

Jenis kerentanan di mana penyerang dapat mengakses atau memanipulasi data berdasarkan reference yang dikirim oleh klien.

Total: 3 Lab
